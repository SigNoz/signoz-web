# Table

A Table is a plot chart that shows the data in a table. 

## Data Formats

### Supported signals

- Logs
- Traces
- Metrics

This panel type supports any time series data. The time series data can be from logs, traces, or metrics.

## Configuration

### Column Units

When you want to format the column values in human readable format, you can use the column units. The column units are the units of the column values.
