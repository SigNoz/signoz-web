## Prerequisites

### 1. Set up a Local Kubernetes Cluster

Choose one of the following options to set up your local Kubernetes cluster:

#### Option 1: Minikube
- Follow the [official Minikube installation guide](https://minikube.sigs.k8s.io/docs/start/)
- Recommended configuration for SigNoz:
  ```bash
  minikube start --memory=8g --cpus=4
  ```

#### Option 2: Kind
- Follow the [official Kind installation guide](https://kind.sigs.k8s.io/docs/user/quick-start/)
- Recommended configuration for SigNoz as `values.yaml`:
  ```yaml
  kind: Cluster
  apiVersion: kind.x-k8s.io/v1alpha4
  nodes:
  - role: control-plane
    extraPortMappings:
    - containerPort: 8080
      hostPort: 8080
  ```
- Run the Following command to spin up the k3s cluster:
```bash
k3s create cluster --config values.yaml -n <NAMESPACE>
```

#### Option 3: K3s
- Follow the [official K3s installation guide](https://docs.k3s.io/installation)

### 2. Install kubectl
- Follow the [official kubectl installation guide](https://kubernetes.io/docs/tasks/tools/)
