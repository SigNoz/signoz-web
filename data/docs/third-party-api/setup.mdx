## Setup

If you are using auto instrumentation agents, make sure the following attributes are added

It uses span attributes like
- `net.peer.name`
- `http.url`
- `http.target`
to automatically derive API calls.

in case of manual instrumentation, ensure that all the client to server calls are instrumented and the above attributes are added
