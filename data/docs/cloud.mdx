## Popular Use Cases

### Application Performance Monitoring

Get out-of-the-box charts for key application metrics like latency(p99, p90, and p50), ops per second, error rate, Apdex, top operations, etc. with SigNoz APM. You can also monitor database and external calls being made by your application. Instrument your application with OpenTelemetry and start sending data to SigNoz within minutes.

Distributed tracing is critical for microservices-based applications. Our APM module comes bundled with traces too. You can jump from application metrics to traces with just a click.

Follow these instructions based on the programming language of your application:

- [Python](https://signoz.io/docs/instrumentation/python/)
- [Java](https://signoz.io/docs/instrumentation/java/)
- [Javascript](https://signoz.io/docs/instrumentation/javascript/)
- [Golang](https://signoz.io/docs/instrumentation/golang/)
- [PHP](https://signoz.io/docs/instrumentation/php/)
- [.NET](https://signoz.io/docs/instrumentation/dotnet/) 
- [Ruby on Rails](https://signoz.io/docs/instrumentation/ruby-on-rails/)
- [Elixir](https://signoz.io/docs/instrumentation/elixir/)
- [Rust](https://signoz.io/docs/instrumentation/rust/)
- [Swift](https://signoz.io/docs/instrumentation/swift/)

### Log Management

Ingest logs from any source and use SigNoz as the centralized log management solution. Quickly search and filter through your logs using our query builder. Create charts, add them to dashboards and create alerts on critcal metrics from logs.

[Start sending logs to SigNoz](https://signoz.io/docs/userguide/logs/)

### Metrics & Dashboards

Send any kind of metrics to SigNoz and create customized dashboards to monitor them. Create visualizations that suit your needs by using the wide range of panel options we provide - time-series, value, table, list, bar chart, pie chart, etc. You can use our easy to use query builder to query your metrics. We also support PromQL for users with a Prometheus background and Clickhouse queries(our database) for advanced use cases.

[Send metrics to SigNoz](https://signoz.io/docs/userguide/send-metrics-cloud/)

### Monitor your Host machines or VM

Monitor your host machine easily with Infrastructure Monitoring. Get started within minutes to get deep visibility into your host machines with metrics for:

- CPU usage
- Memory usage
- Disk I/O
- Filesystem usage
- Network I/O

and more.

[Learn how to send host metrics to SigNoz](https://signoz.io/docs/userguide/hostmetrics/)

### Kubernetes Infra Metrics & Logs Monitoring

Get insights into the health and performance of your Kubernetes cluster. Monitor key metrics like CPU and memory usage, network I/O, filesystem usage, etc. for your pods and nodes.
